## Description
Backend application for the Winnipeg Music Festival Registration platform.  Built using Nest.js, Graphql, and Postgres.  Made to work together with 'wmf-nuxt'

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
